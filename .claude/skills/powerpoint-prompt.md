# Skill: PowerPoint-Prompt erstellen

## Wann laden
Wenn ein PowerPoint-Prompt fuer einen Schulungstag erstellt oder aktualisiert werden soll.

## Vorgehen
1. Theorie-Datei des Tages lesen (Schulung/Tag-*/01-Theorie*.md)
2. Aufgaben-Datei lesen (Schulung/Tag-*/02-Aufgabe*.md)
3. Folienstruktur ableiten:
   - Titelfolie + Agenda
   - 1 Folie pro Hauptabschnitt der Theorie
   - Tabellen und Diagramme wo im Text Tabellen/Listen stehen
   - Uebungsfolien mit Schritten aus der Aufgabe
   - Zusammenfassung + Key Takeaways
4. Pro Folie: Titel, Inhalt (max. 5-6 Bulletpoints), Diagramm-Beschreibung

## Design-Vorgaben
- Farbschema: Dunkelblau (#1a365d), helles Grau (#f7fafc), Akzent Orange (#ed8936)
- Min. 24pt Body, 36pt Titel
- Wenig Text, viele Diagramme/Tabellen
- Sprache: Deutsch

## Output
Markdown-Datei in Schulung/PowerPoint-Prompts/Tag-X-Prompt.md
